# Student Dashboard

A simple student dashboard webpage created using HTML and CSS Grid.  
This dashboard displays student courses, marks, and notifications in a clean and responsive layout.

## Features

- Course details table
- Marks section
- Notifications section
- Responsive CSS Grid layout

## Technologies Used

- HTML5
- CSS3 (CSS Grid)

## How to View

Download or clone this repository, then open the file `index.html` in a browser (Chrome, Firefox, or Edge).
